

## 📌 Overview

**Smart Indian AI Election Assistant** is an AI-powered web application designed to help voters with election-related information in a simple and interactive way.
It allows users to verify voter eligibility, access candidate details, and interact in multiple regional languages.

---

## 🚀 Features

* 🗳️ Voter Eligibility Check (via Voter ID)
* 🤖 AI Chat Assistant for election queries
* 🌐 Multi-language Support (Hindi, English, Marathi, Punjabi, Chhattisgarhi)
* 🧑‍💼 Candidate Information Access
* 📷 Face Verification (for identity confirmation - demo based)
* 📊 User-friendly and interactive UI

---

## 🛠️ Tech Stack

* **Frontend:** React.js / Vite
* **Backend:** Node.js / Express.js
* **Database:** MongoDB
* **AI Integration:** OpenAI API
* **Authentication:** Face Recognition (Demo)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/Smart-Indian-Election-AI.git
cd Smart-Indian-Election-AI
```

### 2️⃣ Install dependencies

#### Frontend:

```bash
cd frontend
npm install
npm run dev
```

#### Backend:

```bash
cd backend
npm install
npm start
```

## 🌐 Usage

1. Open the application in your browser
2. Select your preferred language
3. Enter your Voter ID to check eligibility
4. Use AI Chat Assistant for queries
5. (Optional) Use face verification for demo authentication
